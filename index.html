Arctic Design Kit

Building modern and efficient UI components with the best tools.

Key Features

Zero-runtime CSS-in-JS

Based on @pigment-css/react which extracts styles to CSS files at build time, ensuring zero-runtime overhead.

React Server Components

Compatible with React Server Components (RSC) for seamless server-side rendering.

Floating UI

Position floating elements effortlessly with @floating-ui/react for tooltips, popovers, and dropdowns.

Interactive Animations

Framer Motion makes it easy to create fluid animations. Its a modern animation library for JavaScript and React.

Capabilities

Snow

Arctic Design's Snow library, or @arctic-kit/snow, is a React component library offering essential UI components for streamlined development and consistent application design.

Iceberg

The Iceberg project is an AI-powered form builder designed to revolutionize form creation and management within web applications. Currently in development.

Aurora

The Aurora project is an AI-powered WYSIWYG editor that aims to revolutionize content creation and editing within the Arctic Design ecosystem. Currently under development.
