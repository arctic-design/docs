Core Fundamentals

Zero Runtime CSS

Based on @pigment-css/react which extracts styles to CSS files at build time, ensuring zero-runtime overhead.

RSC Compatible

Compatible with Next.js React Server Components (RSC) for seamless server-side rendering.

Floating UI

Position floating elements effortlessly with @floating-ui/react for tooltips, popovers, and dropdowns.

Interactive Animations

Framer Motion makes it easy to create fluid animations in React.

Capabilities

Snow

Arctic Design's Snow library, or @arctic-kit/snow, is a versatile React component library offering essential UI components for streamlined development and consistent application design. These components are meticulously crafted for seamless integration, providing both flexibility and ease of use.

Iceberg

The Iceberg project is an AI-powered form builder designed to revolutionize form creation and management within web applications. Currently in development, Iceberg will provide intelligent features to simplify and enhance the form-building process

Aurora

The Aurora project is an AI-powered WYSIWYG editor that aims to revolutionize content creation and editing within the Arctic Design ecosystem. Currently under development, Aurora promises a range of innovative features to enhance the editing process
