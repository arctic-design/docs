Foundations of Arctic Design

The Foundations of Arctic Design lay the groundwork for a cohesive and scalable design system. These foundations include essential elements such as:

  • Colors
  • Icons

Each aspect is carefully structured to ensure consistency and flexibility across all UI components. Arctic Design prioritizes accessibility, modern aesthetics, and performance, providing a solid base for building intuitive and visually appealing user experiences. These core principles support seamless collaboration between design and development, enabling scalable and adaptable design across various projects.

Colors
The Arctic Design system's color palette is structured into distinct categories using `ColorType`, which includes primary, secondary, success, info, warning, error, and grey, each with a range of shades from light to dark (e.g., 50 to 900). Additionally, the system features a `NeutralColorType` with shades from 0 (lightest) to 1000 (darkest), providing a flexible yet cohesive visual identity for UI elements across the design system.
Icons
The Arctic Design Kit uses `@arctic-kit/icons` for all its icon needs. This package provides a collection of SVG icons that seamlessly integrate with React components, enhancing UI and visual communication. Each icon comes with a default `className` of `arctic-icon-medium`, allowing easy application of global styles like width and height through CSS.
