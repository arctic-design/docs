## Installation Guide

Integrate the `@arctic-kit/snow` library into your project by following these steps. This guide covers installation via both `yarn` and `npm`, configuring styles, setting up fonts, and using a component.

### 1. Install

Choose your preferred package manager to install the library along with the dependent libraries.

- **Using Yarn:**

<CodeBlockSection>{`yarn add @arctic-kit/snow @floating-ui/react @pigment-css/react framer-motion`}</CodeBlockSection>

- **Using NPM:**

<CodeBlockSection>{`npm install @arctic-kit/snow @floating-ui/react @pigment-css/react framer-motion`}</CodeBlockSection>

### 2. Fonts Setup

To ensure the components render correctly, include the necessary font files in your project. Instead of using `<link>` tags, we'll use the `@fontsource` package to import the fonts directly into your styles:

- **Install Fontsource Package:**

  - **Using Yarn:**

  <CodeBlockSection>{`yarn add @fontsource/inter`}</CodeBlockSection>

  - **Using NPM:**

  <CodeBlockSection>{`npm install @fontsource/inter`}</CodeBlockSection>

### 3. Configuration

#### **For Vite:**

- Install `@pigment-css/vite-plugin` as a development dependency:

  <CodeBlockSection>{`yarn add -D @pigment-css/vite-plugin`}</CodeBlockSection>

- Then in your `vite.config.ts`, set up the following configuration:

<CodeBlockSection>
{
`
import { defineConfig } from 'vite';
import react from '@vitejs/plugin-react';
import { extendTheme, pigment } from '@pigment-css/vite-plugin';
import { createDefaultTheme } from '@arctic-kit/snow';

const defaultTheme = createDefaultTheme();
const theme = extendTheme({
...defaultTheme,
cssVarPrefix: 'snow',
});

export default defineConfig({
  plugins: [
    pigment({
      theme,
    }),
    react(),
  ],
});
`}

</CodeBlockSection>

- In your root file (e.g., `main.ts`), add:

<CodeBlockSection>
{
`
  
import '@pigment-css/react/styles.css';
import '@arctic-kit/snow/style.css';

`
}

</CodeBlockSection>

#### **For Next.js (App Router):**

- Install the required dependencies:

  <CodeBlockSection>{`yarn add @pigment-css/nextjs-plugin`}</CodeBlockSection>

- In your `next.config.mjs`, configure the following:

<CodeBlockSection>
{
`
import { withPigment, extendTheme } from '@pigment-css/nextjs-plugin';
import { createDefaultTheme } from '@arctic-kit/snow';

const defaultTheme = createDefaultTheme();
const theme = extendTheme({
...defaultTheme,
cssVarPrefix: 'snow',
});

export default withPigment({...nextConfig}, {
    theme,
  });
`}

</CodeBlockSection>

- In your root layout file (e.g., `layout.tsx`), add:

<CodeBlockSection>
{
`
  
import '@pigment-css/react/styles.css';
import '@arctic-kit/snow/style.css';

`
}

</CodeBlockSection>
